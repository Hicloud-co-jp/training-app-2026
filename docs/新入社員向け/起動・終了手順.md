# 社員管理アプリ 起動・終了手順

環境構築が終わった後に毎日使う手順です。操作にはPowerShellを使用します。Ubuntuは使用しません。

## 起動する

1. WindowsのスタートメニューからDocker Desktopを開く
2. 「Engine running」と表示されるまで待つ
3. PowerShellを開く
4. 次のコマンドを実行する

```powershell
cd C:\mywork\training-app-2026
docker compose up -d
```

[http://localhost:8080](http://localhost:8080)を開きます。社員一覧が表示されたら起動完了です。

## 終了する

PowerShellで次のコマンドを実行します。

```powershell
cd C:\mywork\training-app-2026
docker compose stop
```

終了後はPowerShellとDocker Desktopを閉じて構いません。`stop`では社員データは消えません。

## 起動状態を確認する

```powershell
cd C:\mywork\training-app-2026
docker compose ps
```

`laravel.test`と`mysql`が表示されていれば正常です。

## 画面が開かないとき

Docker Desktopが「Engine running」になっているか確認してから、次を実行します。

```powershell
cd C:\mywork\training-app-2026
docker compose ps
docker compose logs --tail=30 laravel.test
```

実行したコマンドと表示内容を研修担当者へ伝えてください。パスワードやtokenは貼らないでください。

## 実行しないコマンド

次のコマンドは研修用DBデータを削除します。

```powershell
docker compose down -v
```

研修担当者から指示された場合以外は実行しないでください。
