# 社員管理アプリ2026 資料一覧

資料を対象者別に分けています。新入社員は「新入社員向け」、研修担当者は「講師向け」から確認してください。

## 新入社員向け

1. [環境構築](./新入社員向け/環境構築.md) — PowerShellとDocker Desktopによる初回構築
2. [起動・終了手順](./新入社員向け/起動・終了手順.md) — 毎日の起動、確認、終了、更新方法
3. [初級課題](./新入社員向け/課題/初級課題.md) — 入社年数、検索条件保持、ソート、バリデーションを学ぶ
4. [中級課題](./新入社員向け/課題/中級課題.md) — 住所検索画面の作成と外部API連携を学ぶ
5. [上級課題](./新入社員向け/課題/上級課題.md) — DB正規化、CSV出力、画像アップロードを学ぶ

## 講師向け

1. [技術スタック・設計・フォルダ構成](./講師向け/技術スタック・設計・フォルダ構成.md) — 現在の実装、処理フロー、DB、主要ファイル
2. [初級・中級・上級課題対応表](./講師向け/初級・中級・上級課題対応表.md) — 課題配置、開始branch、指導上の注意
